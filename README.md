# MES決策室

連結生產計畫與現場執行的核心系統，即時掌控工單、設備、品質與物料狀態。

- 原始系統：MES（Manufacturing Execution System）
- 產業分類：智慧製造｜製造管理
- 使用部門：生產、品保、製造工程、設備、生管
- 工作流程：工單接收 → 現場派工 → 製程執行與數據採集 → 品質判定 → 完工入庫回饋
- AI 應用：AI異常製程參數即時預警、機器學習輔助良率異常根因分析、智慧產能瓶頸辨識
- 主要 KPI：良率、稼動率、在製品週轉時間、首件合格率
- 代表產品：Siemens Opcenter（原Camstar）、Rockwell FactoryTalk、SAP ME、鼎捷e-MES
